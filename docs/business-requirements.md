# ビジネス要望書 — 補助金・助成金マッチングサイト

## 1. プロジェクト概要

**補助金・助成金のマッチングサイト**を構築する。
企業が自社に適用できる補助金・助成金を簡単に見つけられるサービス。

## 2. ビジネスモデル

```
ユーザー（中小企業経営者）
  → 一問一答形式で会社情報を入力
  → マッチする補助金・助成金リストを表示
  → 「詳しい相談はこちら」→ リード獲得
  → ベンダー（Growing up = 俺たち）がシステム開発案件として受注
```

**収益構造:**
- 補助金マッチング → 無料（リード獲得が目的）
- 「IT導入補助金」「ものづくり補助金」等でシステム開発が補助対象
- 補助金を使ってシステム開発を受注 → ベンダー大儲け
- 補助金申請サポート → 追加課金も可能

## 3. 参考サイト

### j-izumi.com（情報の泉）
- **良い点:** 20,000件の補助金データ、AIマッチング、申請代行連携
- **悪い点（Naoki評価）:** UIUXがゴミ、わかりにくい
- **機能:** 企業登録 → マッチング → 通知 → 申請代行

### 俺たちがやる差別化

| j-izumi | うちのサイト |
|---|---|
| AI感バリバリ | **AI感を出さない。官公庁風。** |
| 複雑なUI | **一問一答形式で迷わない** |
| SaaS月額課金 | **無料でリード獲得 → 開発受注** |
| 営業ツール色が強い | **「あなたの会社でもらえるかも」という安心感** |

## 4. デザイン方針

### トンマナ: 官公庁風
- **参考:** 経済産業省、中小企業庁、J-Net21、ミラサポplus
- 色: 紺（#003366）+ 白 + 薄いグレー
- フォント: 游ゴシック or Noto Sans JP（Regular/Medium）
- ロゴ感: 省庁の紋章的な信頼感
- 装飾: 最小限。情報密度は高め。過度なアニメーションなし
- 写真: スーツのビジネスマン、オフィス、書類（フリー素材OK）
- **「ちょっと古いけど、信頼できる」**

### NG
- AI、ロボット、チャットボット感を出さない
- 派手なグラデーション、ネオン色
- 「AIが自動で！」みたいなコピー
- ポップすぎるUI

## 5. 機能要件

### 5.1 一問一答マッチングフロー（コア機能）

**UXの核心: サンクコスト効果**

進捗バーを表示し、途中離脱を防ぐ。「ここまで入力したのにもったいない」と思わせる。

```
[==============================] 80% 完了！あと2問です

Q1. 会社の所在地は？ → 都道府県選択
Q2. 業種は？ → プルダウン（大分類→中分類）
Q3. 従業員数は？ → 1-5名 / 6-20名 / 21-50名 / 51-100名 / 101名以上
Q4. 資本金は？ → 選択式
Q5. 現在の課題は？（複数選択可）
   □ 設備投資がしたい
   □ IT化・DXを進めたい ← ★ここが刺さればシステム開発受注
   □ 人材を採用したい
   □ 海外展開を考えている
   □ 研究開発をしたい
   □ 事業承継を考えている
Q6. 年間売上規模は？ → 選択式
Q7. これまで補助金を申請したことは？ → はい/いいえ
Q8. メールアドレス → ★リード獲得
```

**結果画面:**
```
あなたの会社に該当する可能性がある補助金・助成金

🏛️ IT導入補助金（最大450万円）
   ✅ 適合度: 高
   概要: ITツール導入費用の1/2〜3/4を補助
   申請期限: 2026年6月30日
   → 詳しく見る

🏛️ ものづくり補助金（最大1,250万円）
   ✅ 適合度: 中
   ...

💬 「この補助金を活用してシステム開発しませんか？」
   → 無料相談はこちら（← リード獲得フォーム）
```

### 5.2 WordPress ベース

- WordPress + カスタムテーマ
- 一問一答は JavaScript（React or Vanilla JS）で実装
- 補助金データはカスタム投稿タイプ（Custom Post Type）
- マッチングロジックはPHP or REST API

### 5.3 管理画面

- 補助金データの CRUD（タイトル、金額、対象業種、対象地域、申請期限等）
- リード一覧（メールアドレス + 回答内容）
- マッチング結果の閲覧
- 補助金データの一括インポート（CSV）

### 5.4 その他

- レスポンシブ（スマホ対応必須 — 経営者はスマホで見る）
- SSL必須
- OGP設定（SNSシェア用）
- Google Analytics / Tag Manager 連携
- お問い合わせフォーム（Contact Form 7 or カスタム）
